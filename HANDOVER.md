# 🧧 복(지)주머니 프로젝트 인수인계서 (Handover)

이 문서는 **안티그래비티(AI)**와 **사용자(총괄 디렉터)**가 함께 빌드해온 **'복(지)주머니'** 프로젝트를 팀원분들께 인수인계하기 위해 작성되었습니다.

---

## 1. 프로젝트 개요
사용자의 개인 상황을 분석하여 **숨은 정부 지원금**을 찾아주고, 그 과정에서 혜택을 '복주머니'에 담아주는 듯한 경험을 제공하는 서비스입니다. 기존 명칭 '웰페어 스코어'에서 보다 친근한 **'복(지)주머니'**로 리브랜딩되었습니다.

## 2. 핵심 구현 기능 (V20+ 기준)

### ✅ 복지 점수 산출 로직
- **5단계 설문**: 연령, 가구 상황, 거주 지역, 소득, 관심 분야를 입력받습니다.
- **실시간 점수**: 입력 데이터에 기반해 시급성과 적합도를 계산하여 점수(0~100점)를 보여줍니다.

### ✅ 데이터 매칭 엔진
- **중앙정부 데이터**: `generated_data.js`에 약 수천 건의 복지 데이터 내장.
- **지역 밀착형 데이터**: 전국 17개 광역자치단체 청년 포털 데이터를 직접 연동 (`script.js` 내 `REGIONAL_PORTALS`).
- **자립준비청년 특화**: '자립준비청년' 선택 시 전용 지원금 및 포털이 최상단에 노출되도록 설계.

### ✅ AI 챗봇 '로거(Logger)'
- **페르소나**: 귀여운 골든 리트리버 강아지 (헤드셋 착용).
- **자유 검색**: "20대 주거지원" 등 키워드 입력 시 관련 혜택을 즉시 필터링하여 카드 형태로 제공.
- **블로그 연동**: 검색된 혜택 카드를 클릭하면 바로 운영 중인 티스토리 블로그의 관련 글로 연결됩니다.

### ✅ UI/UX 최적화
- **모바일 퍼스트**: 모바일 기기에서 버튼 정렬, 챗봇 스크롤 등 모든 불편 사항을 수정 완료.
- **수익화 설계**: '신청하기' 버튼 옆에 '블로그 꿀팁' 버튼을 배치하여 블로그 유입 극대화.

---

## 3. 기술 스택 및 환경
- **Frontend**: Pure HTML, CSS, JavaScript (프레임워크 없이 빠르고 가볍게 구현).
- **Data Processor**: Python (정부 API 데이터를 가공하여 JS 데이터로 변환).
- **Deployment**: Vercel을 통한 자동 배포 환경.
- **Version Control**: Git이 초기화되어 있으며, GitHub 협업 준비 완료.

## 4. 폴더 구조 설명
- `/index.html`: 메인 인터페이스 및 설문 단계 정의.
- `/script.js`: 점수 계산 로직, 챗봇 엔진, 지역 데이터 포함 (가장 핵심).
- `/style.css`: 전체 디자인 테마 및 모바일 반응형 스타일.
- `/generated_data.js`: 중앙정부 복지 데이터베이스.
- `/data-engine/`: 데이터 수집(Fetcher) 및 가공(Processor) 스크립트 모음.
