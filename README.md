# 🧧 복주머니 (BokJumoney)

> **"나만 몰랐던 숨겨진 지원금, 복주머니에서 바로 찾아드려요!"**

**복주머니**는 사용자의 연령, 가구 형태, 소득 수준 등을 분석하여 정부 및 지자체에서 제공하는 숨은 복지 혜택과 지원금을 실시간으로 매칭해 주는 프리미엄 맞춤형 서비스입니다. 2026년 최신 데이터를 반영하여 가장 정확한 정보를 제공합니다.

---

## ✨ 주요 기능

- **🎯 맞춤 혜택 정밀 진단**: 3분 내외의 간단한 설문으로 나에게 딱 맞는 지원금을 즉시 추출합니다.
- **🧧 숨겨진 '복' 카운팅**: 추상적인 점수 대신, 내가 실제로 받을 수 있는 혜택의 개수를 직관적으로 보여줍니다.
- **🐶 AI 챗봇 '로거' 연동**: 강아지 아바타 '로거'와 대화하며 궁금한 복지 혜택을 자유롭게 질문하고 답변을 받을 수 있습니다.
- **📍 지역별 특화 매칭**: 전국 17개 광역자치단체 및 기초지자체의 전용 포털 데이터를 연동하여 지역 밀착형 혜택까지 빠짐없이 챙겨줍니다.
- **📖 상세 신청 가이드**: 발견된 혜택의 신청 방법과 꿀팁을 연동된 블로그를 통해 상세히 안내합니다.
- **📥 PDF 리포트 제공**: 진단 결과를 PDF 파일로 저장하여 필요할 때마다 꺼내 볼 수 있습니다.

## 🎨 디자인 포인트

- **Premium Luck Theme**: 따뜻한 Coral Red와 Gold 톤의 디자인으로 '복이 들어온다'는 긍정적인 경험을 선사합니다.
- **Interactive UI**: 로딩 중의 복주머니 애니메이션, 결과 창의 '찾았다!!' 뱃지 등 역동적인 인터랙션을 제공합니다.
- **최적화된 모바일 경험**: 모바일 기기에 최적화된 레이아웃과 버튼 배치를 통해 언제 어디서나 편리하게 이용 가능합니다.

## 🛠 기술 스택

- **Frontend**: HTML5, Vanilla CSS, JavaScript (ES6+)
- **Data Engine**: Python (API Fetcher & Data Merging Pipeline)
- **Deployment**: Vercel / Static Web Hosting

## 📂 프로젝트 구조

- `index.html`: 앱의 전체 구조 및 설문 인터페이스 정의
- `script.js`: 매칭 알고리즘, 애니메이션 제어 및 AI 챗봇 인터랙션 로직
- `style.css`: 프리미엄 디자인 시스템 및 애니메이션 라이브러리
- `generated_data.js`: 중앙정부 및 지역별 복지 데이터베이스 (JSON 기반)
- `data-engine/`: 데이터 수집(Python) 및 가공 스크립트 모음
